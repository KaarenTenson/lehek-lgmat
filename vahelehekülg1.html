Aritmeetilised tehted funktsioonidega. Liitfunktsioon

Olgu antud funktsioonid f : X1 → R ja g : X2 → R. Olgu X = X1 ∩ X2.

1. Funktsioonide f ja g summaks nimetatakse funktsiooni f + g : X → R, mis defineeritakse võrdusega

(f + g)(x) = f(x) + g(x) iga x ∈ X korral.

2. Funktsioonide f ja g vaheks nimetatakse funktsiooni f − g : X → R, mis defineeritakse võrdusega

(f − g)(x) = f(x) − g(x) iga x ∈ X korral.

3. Funktsioonide f ja g korrutiseks nimetatakse funktsiooni fg : X → R, mis defineeritakse võrdusega

(fg)(x) = f(x)g(x) iga x ∈ X korral.

4. Funktsioonide f ja g jagatiseks nimetatakse funktsiooni f/g: X \ {x: g(x) = 0} → R, misdefineeritakse võrdusega

(f/g)(x) = f(x) g(x)iga x ∈ X \ {x | g(x) = 0} korral.
